Vérification Google Search Console

Ce fichier permet de vérifier la propriété du site RAWRAGENCY.fr dans Google Search Console.

Pour configurer :

  1. Allez sur Google Search Console
  2. Ajoutez votre propriété rawragency.fr
  3. Choisissez la méthode "Balise HTML" ou "Fichier HTML"
  4. Remplacez le contenu de ce fichier par le code fourni par Google
